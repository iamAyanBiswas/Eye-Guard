# Eyeguard ML Model Output

This folder stores the trained model files generated by the Colab training script.

## Expected Files

After running `training-ml/train_fatigue_model.py` on Google Colab, download and place these files here:

| File | Description |
|------|-------------|
| `fatigue_classifier.h5` | The trained Keras neural network (21 inputs → 4 fatigue classes) |
| `feature_scaler.pkl` | The StandardScaler used to normalize input features |

## Usage

The `api/app.py` server loads these files automatically on startup to serve predictions.
